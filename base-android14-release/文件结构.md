# Android 14 源码文件结构

## 根目录
- 构建配置: Android.bp, Android.mk, CleanSpec.mk, PREUPLOAD.cfg
- 所有权管理: 各种 OWNERS 文件和 BATTERY_STATS_OWNERS 等特定模块所有者文件
- 许可证和元数据: MODULE_LICENSE_APACHE2, NOTICE, METADATA
- 测试配置: TEST_MAPPING

## 主要目录

### apex/
Android Pony EXpress (APEX) 模块，包含系统组件如 blobstore 和 jobscheduler

### api/
API 定义和文档生成相关文件

### boot/
启动镜像配置，包括隐藏 API 设置和预加载类

### cmds/
系统命令行工具集合，如 am(活动管理), pm(包管理), wm(窗口管理) 等

### config/
系统配置文件，包括启动配置和版权头等

### core/
核心框架代码，包含 Java API 和 JNI 实现
- java/: Java 框架代码
- jni/: JNI 本地代码实现
- res/: 核心资源文件

### data/
系统数据文件
- etc/: 系统配置 XML 文件
- fonts/: 系统字体文件
- keyboards/: 键盘映射文件
- sounds/: 系统音频文件
- videos/: 系统视频文件

### docs/
文档相关文件

### graphics/
图形系统相关代码

### libs/
各种系统库
- hwui/: 硬件加速 UI 库
- androidfw/: Android 框架库
- input/: 输入系统库

### media/
媒体框架代码

### services/
系统服务实现
- core/: 核心服务
- backup/: 备份服务
- accessibility/: 无障碍服务

### native/
原生代码库

### wifi/
WiFi 相关代码(已移动)
