# Android系统输入法相关代码位置

## 核心服务代码
- **输入法管理服务**: `/services/core/java/com/android/server/inputmethod/InputMethodManagerService.java`
  - 系统级输入法管理服务，负责管理输入法的绑定、切换、显示/隐藏等核心功能

- **输入法接口定义**: `/core/java/android/view/inputmethod/InputMethod.java`
  - 定义了输入法接口，所有输入法实现都需要实现此接口

- **输入法信息类**: `/core/java/android/view/inputmethod/InputMethodInfo.java`
  - 描述输入法组件的元信息，包括包名、服务名、子类型等

## 相关测试代码
- **压力测试**: `/tests/InputMethodStressTest/`
- **服务测试**: `/services/tests/InputMethodSystemServerTests/`
- **核心测试**: `/services/tests/servicestests/src/com/android/server/inputmethod/`

## 设置相关
- **设置库**: `/packages/SettingsLib/src/com/android/settingslib/inputmethod/`
  - 包含输入法设置相关的Preference组件和工具类

## 核心框架测试
- **核心测试**: `/core/tests/coretests/src/android/view/inputmethod/`
  - 包含输入法相关的各种单元测试

## Proto定义
- **Proto文件**: `/core/proto/android/view/inputmethod/`
  - 包含输入法管理器和输入连接相关的Proto定义

## 内部接口定义
- **内部输入法接口**: `/core/java/com/android/internal/inputmethod/`
  - 包含内部使用的输入法相关接口和工具类
